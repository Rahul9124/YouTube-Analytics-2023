# YouTube-Analytics-2023
This project provides a detailed exploratory data analysis (EDA) of the most-subscribed YouTube channels in 2023. Using a dataset of nearly 1,000 top-tier creators, this analysis uncovers the underlying patterns of digital success, geographic distribution, and financial performance in the modern creator economy.

Global YouTube Statistics 2023 - Data Analysis

  Project Overview
        
       This project performs an in-depth exploratory data analysis (EDA) on the top 995 YouTube channels of 2023. Using Python, 
       I explored trends in subscribers, video views, earnings, and geographic distributions to understand what makes a channel successful in the current digital landscape.

  Key Features

    Data Cleaning         : Handled missing values and formatted large numerical datasets for analysis.
    Exploratory Analysis  : Visualized correlations between subscriber counts and total video views.
    Earnings Insight      : Analyzed monthly and yearly earnings across different channel categories.
    Sentiment Analysis    : Analyzed the sentiment of video titles to see if positivity impacts reach.

  Technologies Used

    Python                : Jupyter Notebook
    Pandas & NumPy        : Data Manipulation
    Matplotlib & Seaborn  : Static Visualizations
    Plotly Express        : Interactive Charts

  How to Run

    Clone this repository.
    Ensure you have the `YouTube_Statistics.csv` file in the same directory as the notebook.
    Install dependencies: `pip install pandas numpy matplotlib seaborn plotly textblob`
    Open `YouTube_Analytics_2023.ipynb` in Jupyter or VS Code and run all cells.

  Visualizations
    
    The notebook includes interactive charts showing top-performing categories (Music, Entertainment, Education) and geographic hotspots for content creation.
